# gossadmin

shadowsocks server write in golang.

features:

multi user, multi port.

account manager.

web ui manager.

net traffic limited.

gossadmin -help

gossadmin -config Gen config file

# Install

go get github.com/yetist/gossadmin

# Usage

- create database
- write config file, run `gossadmin -config` to see example.
- ./gossadmin
